What is GPCreator?

GPCreator app is a compressor app with lots of tricks and multitasking tools.
This app allows you to compress files in multiple tabs with just one click.
This application will act as a file manager to keep your files organized.
You no longer need to explore your hard drive pysically to find files when you want to compress.
Doing that is so tiring. Too many windows on your desktop can be irritating and confusing sometimes.
GPCreator offers you the option to open many folders and create a new tab for each selected folder.
You can choose the files you want from different directories to export the items you selected
to different tabs and compress them.
You can choose and manage the files you want to archive them with ease.
You can also open multiple archive files with just one window.
Opening archive files one by one with an app is so confusing sometime
and messy on your desktop, but in GPCreator,
you can open them all and view all the files in one strip in each tab in the main GPCreator window so
that you can see all the files easily. With this, you no longer need to click and open the folders
one by one in your archive files when you are looking for something.
This app is also very useful when you want to search for multiple files in a folder or in archive files.
This application can also rearrange your documents so that they can be found fasterwhen you needed.
You can search for multiple files in folders or in an archive file without opening the folder or the
archive file and extract only the files you searched for in the archive file.


Features of GPCreator:

• Create multiple archives simultaneously with ease.

• Drag and drop support.

• Shut down your computer using GPCSleep Timer Tool. Set the duration to a minimun of 1 minute and your computer will shutdown automatically.
 
• Add Mini Wizard Tool - allows you to compress the files and folders you have selected on your hard drive. Even if you have stopped the compressing process, you can recompress them again just by clicking the repack menu in the commands menu of the Add Wizard Tool Window.

• Shred Mini Wizard Tool and GPCreator Shredding files/folders - secure deletion that prevents file recovery.

• Import/Export Project - You can save your work and continue it again. With this feature you have no worries anymore to manage your documents.

• Lock/Unlock mode - You can lock your tabs so that your project name does not change when you open a folder to the currently selected tab.

• Search Multiple Files Tool - This tool allows you to search files with different strings simultaneously.

• Multiple passwords - You can set multiple passwords in each tab for your archive files.

• Self Compress Tool - You can compress all the tabs individually without interfering with the GPCreator app. This feature allows you to compress  files with different archive formats.

• Create in Hard Drive Tool - Organize your messy random files in just one click.

• Archive SearchTract Tool - helps you find things fast. Its unique features allow it to dig out information in even the most obscure file formats. You can search files in ZIP, RAR, CAB, 7-Zip, Bzip, CPIO, XZ, GZIP, ISO, LZH, TAR without opening these archive formats, and Archive SearchTract Tool will show you all the results and can extract only the files you searched for. You can also use this tool in folders as well. With this tool, you can search files with ease.

• ...And many more tricks, new tools and updates! (For more info please visit our youtube channel)
